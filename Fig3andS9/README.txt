Model for "Dendritic action potentials and computation in human layer 2/3 cortical neurons"
The model replicates Figure 3 and Supplementary 9.

Model usage:
This code was written in and requires NEURON (tested only in version 7.4) which is freely available from http://www.neuron.yale.edu
This model was run under the unix/linux and windows 10 operating system.

Proceed as follows:
- Download and expand this archive and compile the mod files located in the "_mod" folder with nrnivmodl (nrnivmodl _mod/).

- run the demo for unix/linux:
x84/64/special mosinit.hoc

or for windows double click mosinit.hoc
